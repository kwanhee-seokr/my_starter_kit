# Playwright MCP를 활용한 웹 애플리케이션 오류 분석 및 해결

## 개요
Next.js 16 웹 애플리케이션(localhost:3000)에서 발생하는 오류를 Playwright MCP를 통해 수집, 분석, 해결합니다.

---

## 반복 프로세스

### 1단계: 오류 정보 수집
- Playwright MCP 브라우저로 http://localhost:3000 접속
- 콘솔 로그/에러 수집
- 페이지 스크린샷 캡처

### 2단계: 오류 원인 분석
- 수집된 콘솔 에러 메시지 분석
- 관련 소스 코드 확인

### 3단계: 오류 해결
- 식별된 문제 코드 수정

### 4단계: 테스트
- 수정 후 재접속하여 검증

### 5단계: 반복
- 오류가 남아있으면 1단계부터 반복
